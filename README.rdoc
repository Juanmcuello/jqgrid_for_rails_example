== Jqgrid For Rails sample application

This is a <b>Rails 3.2</b> sample application that uses
jqgrid_for_rails[http://github.com/Juanmcuello/jqgrid_for_rails] gem.


== Installation

  $ git clone git://github.com/Juanmcuello/jqgrid_for_rails_example.git

  $ cd jqgrid_for_rails_example

  $ bundle install

  $ bundle exec rake db:create

  $ bundle exec rake db:migrate

  $ bundle exec rails s
